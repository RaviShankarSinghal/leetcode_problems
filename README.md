## LeetCode Problems


# Two Sum Problem
  Example 1:
  Input: nums = [2,7,11,15], target = 9
  Output: [0,1]
  
  Example 2:
  Input: nums = [3,2,4], target = 6
  Output: [1,2]
  
  Example 3:
  Input: nums = [3,3], target = 6
  Output: [0,1]

# Palindrome Number
  Example 1:

  Output: true

  Example 2:
  Input: x = -121
  Output: false

  Example 3:
  Input: x = 10
  Output: false

# Longest Common Prefix
  Example 1:
  Input: strs = ["flower","flow","flight"]
  Output: "fl"

  Example 2:
  Input: strs = ["dog","racecar","car"]
  Output: ""
